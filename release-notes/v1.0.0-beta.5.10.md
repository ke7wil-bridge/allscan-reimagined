# AllScan Reimagined v1.0.0 Beta 5.10

Beta 5.10 contains the complete Beta 5.9 release and the verified post-5.9
fixes intended for Beta 6.

## Changes

- Removes the remaining Favorite-node double-click action. A Favorite node
  number now selects the node and closes the menu without connecting it.
- Preserves the checked or unchecked **Disconnect before Connect** preference
  in the same browser and device across page refreshes.
- Moves TGIF daemon credentials out of the readable systemd drop-in and into
  `/etc/allscan-reimagined/connected-clients-daemon.env`, owned by root with
  mode `0600`.
- Preserves and restores the TGIF environment and systemd drop-in during
  installer rollback.
- Reports TGIF credential-file protection status without exposing values.

## Verification

- The UI fixes were verified on KE7WIL, KN4EWT/Netoholics, and Del Webb.
- The complete package installer, packaged PHP and shell lint, migration
  self-test, credential migration, served asset, and companion collector were
  live-verified on KE7WIL on July 21, 2026.
- Existing accounts, passwords, Favorites, node configuration, and bridge
  settings remain preserved by the installer.

Beta 5.10 remains a prerelease. It is not the final Beta 6 release.
