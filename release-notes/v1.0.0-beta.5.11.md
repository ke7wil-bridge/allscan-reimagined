# AllScan Reimagined v1.0.0 Beta 5.11

Beta 5.11 contains the complete Beta 5.10 release and the verified public
four-digit AllStar node-selection correction intended for Beta 6.

## Changes

- Allows legitimate public four-digit AllStar nodes beginning at `2000` to
  populate the **Node #** field when selected from Connection Status.
- Applies the same behavior when selecting a node from Favorites while
  retaining the existing menu-close behavior.
- Continues to exclude private/local nodes `1000` through `1999` from the
  Node Controls selection shortcut.
- Stops hiding legitimate public four-digit nodes from Lookup solely because
  they contain four digits.
- Continues to exclude explicitly configured private bridge nodes from public
  lookup targets.
- Builds release archives without Apple extended-attribute headers.

## Verification

- The selection behavior was verified for public node `2300`, private bridge
  node `1883`, and standard public node `29332` in both Connection Status and
  Favorites.
- The no-version-bump preview was installed successfully on KE7WIL,
  Thomas/KN4EWT, and Del Webb/NY7S before the correction was promoted into
  this release.
- The installer updates both the live webroot and protected ASR master while
  preserving users, passwords, permissions, Favorites, database, node
  settings, bridge settings, friendly names, map cache, and protected
  credentials.

Beta 5.11 remains a prerelease. It is not the final Beta 6 release.
