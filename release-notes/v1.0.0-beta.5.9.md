# AllScan Reimagined v1.0.0 Beta 5.9

Beta 5.9 prevents the optional ASR bridge-client collector from creating continuous load when no connected-client source is configured and repairs three Beta 4 compatibility defects reported during outside testing.

## Changes

- Enables `allscan-reimagined-bridge-clients.timer` only when at least one bridge explicitly selects a Local JSON/file or HTTP API client source with a nonempty path or URL.
- Keeps the timer disabled when bridge cards exist but every Client Source is set to Disabled.
- Stops any already-running optional collection during install/reapply when no managed source is configured.
- Makes the collector synchronize the same source-aware timer state after Reimagined Settings changes.
- Replaces the 15-second activation schedule with a one-minute interval after the previous run becomes inactive, preventing immediate catch-up runs after a slow collection.
- Adds a non-overlap lock, a 30-second service timeout, a 25 percent CPU quota, and a 128 MB memory limit.
- Skips the install-time collection and writes an empty managed cache when no managed source is configured.
- Reports the optional collector as `not needed` in diagnostics and Performance Stats when all client sources are disabled.
- Clarifies that Disabled client sources continue using current external `connected-clients.json` data without starting ASR's optional collector.
- Uses `/etc/allscan/favorites.ini` as the single real primary Favorites file and exposes `/allscan/favorites.ini` as a symlink, keeping ASR and stock AllScan on the same data.
- Creates writable `.bak` files beside current Favorites files so stock AllScan can add and delete entries after every ASR reapply.
- Preserves any differing web-root Favorites copy in protected migration storage before replacing it with the primary-file symlink.
- Replaces the KE7WIL-specific initial HTML title with `AllScan Reimagined`; the saved per-node browser title still appears after configuration loads.
- Guards stock AllScan's unauthenticated index status message when no user object exists, without changing authentication or public-access permissions.
- Recognizes ASR's safe three-line `index.php` HTML wrapper as not needing the stock-index guard.
- Keeps the stock-index safety backup root-only and blocked by the existing web-server backup-file rule.
- Periodically repairs Favorites ownership, writable backup files, and the single-source link through the existing integrity service.
- Retains the Beta 5.6 reconnect repair, Beta 5.7 companion-collector maintenance, and Beta 5.8 Favorite label repair.

## Beta 4 tester reports addressed

- The authentication-status request and missing logged-in menu controls were already repaired in Beta 5 and remain covered by the current independent authentication refresh.
- Stock AllScan Favorites backup creation no longer depends on write permission for the hardened ASR web root.
- Fresh and upgraded installations no longer flash the KE7WIL callsign and node number before their saved title loads.
- Unauthenticated stock `index.php` requests no longer generate the reported null-user warning.

## Thomas incident addressed

On Thomas/KN4EWT, all four ASR bridge cards were live-verified with Client Source set to Disabled. The optional ASR timer was therefore unnecessary. Incident evidence showed its 15-second schedule repeatedly launching collections that often required 20-60 seconds, with VPS load reaching approximately 37-47. Disabling and stopping that timer returned load to approximately 1.2-1.3 while the external bridge/client daemons and cached public-status files continued updating.

## Reinstall

```bash
set -e
url="https://github.com/ke7wil-bridge/allscan-reimagined/releases/download/v1.0.0-beta.5.9/allscan-reimagined-1.0.0-beta.5.9.tar.gz"
pkg="/tmp/allscan-reimagined-1.0.0-beta.5.9.tar.gz"
expected="a2c19cf8cdd36e0abd5d93538331b7021f5431158418516f07768b011fd17dd3"
stage="/tmp/asr-beta-5.9-install"

curl -fL "$url" -o "$pkg"
printf '%s  %s\n' "$expected" "$pkg" | sha256sum -c -
rm -rf "$stage"
mkdir -p "$stage"
tar -xzf "$pkg" -C "$stage"
cd "$stage/allscan-reimagined-1.0.0-beta.5.9"
php -l payload/server/asr-api.php
php -l payload/scripts/asr-bridge-clients.php
php payload/scripts/asr-bridge-clients.php --self-test
sh -n payload/scripts/asr-asterisk-read.sh
bash -n payload/scripts/asr-reapply.sh
bash -n payload/scripts/asr-integrity-check.sh
bash payload/scripts/asr-favorites-permissions.sh --self-test
python3 payload/scripts/asr-patch-connected-clients.py --self-test
python3 payload/scripts/asr-patch-allscan-index.py --self-test
bash ./install.sh
```
