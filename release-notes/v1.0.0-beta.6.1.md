# AllScan Reimagined v1.0.0 Beta 6.1

Beta 6.1 is the corrected side-by-side migration prerelease. It contains the
complete Beta 6 feature set and fixes the legacy-overlay backup failure found
during the first Thomas installation test.

## Migration compatibility fix

- Excludes obsolete `astdb.txt.*` symbolic-link aliases from the temporary
  legacy `/allscan/` rollback archive.
- Keeps the real Asterisk database protected and preserved through its
  authoritative location.
- Continues to preserve the canonical Favorites link.
- Keeps strict archive validation in place: unknown links, hard links, special
  files, unsafe paths, oversized files, and files outside the expected webroot
  remain rejected.
- Adds a regression test using the same legacy link pattern found on Thomas.

## Beta 6 features included

- Installs stock AllScan at `/allscan/` and AllScan Reimagined at `/asr/`.
- Preserves users, Favorites, database, settings, bridge configuration, map
  cache, uploaded logos, and protected secrets.
- Adds the DMR Net Bridge card and controls.
- Adds the mobile Help & Instructions page, mobile Configs and Users pages,
  update notices, and the five-version rollback control.
- Retains authoritative SawStat timing, topology counts, bridge-client
  protections, and cached status behavior.

Beta 6.1 remains a prerelease. Installation remains interactive and creates a
rollback backup before changing the live web interfaces.
