# AllScan Reimagined v1.0.0 Beta 5.1

Beta 5.1 is a small theme-consistency patch for Beta 5.

## What Changed

- The desktop **Lookup** button now matches the Bright Side theme controls.
- The desktop **Lookup** button now matches the Deep Ocean theme controls.
- The desktop **Lookup** button now matches the Matrix theme controls.
- Dark Side and ST:ASL behavior are unchanged.

## Install

Copy this complete block into an interactive terminal on the AllStar node. It downloads the release, verifies the exact SHA-256 checksum, extracts it, and starts the installer:

```bash
set -e

url="https://github.com/ke7wil-bridge/allscan-reimagined/releases/download/v1.0.0-beta.5.1/allscan-reimagined-1.0.0-beta.5.1.tar.gz"
pkg="/tmp/allscan-reimagined-1.0.0-beta.5.1.tar.gz"
stage="/tmp/allscan-reimagined-install-beta5.1"
sum="9f6faba40f80991fb7d880b46012e8e5fdfd5e4758bee14f38d3db72eb8817b6"

sudo rm -rf "$stage"
sudo rm -f "$pkg"
curl -fL "$url" -o "$pkg"
echo "$sum  $pkg" | sha256sum -c -

mkdir -p "$stage"
tar -xzf "$pkg" -C "$stage"

cd "$stage/allscan-reimagined-1.0.0-beta.5.1"
bash ./install.sh
```

Do not wrap the final installer in a heredoc or other non-interactive shell. The installer and the official AllScan updater require an interactive terminal.
