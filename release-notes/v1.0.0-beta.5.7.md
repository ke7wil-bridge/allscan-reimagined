# AllScan Reimagined v1.0.0 Beta 5.7

Beta 5.7 adds a controlled daily maintenance restart for the companion connected-client collector after longer observation showed gradual retained-memory growth even though the Beta 5.6 Socket.IO thread leak remained fixed.

## Changes

- Installs `allscan-reimagined-connected-clients-maintenance.service` and its timer only when `connected-clients-daemon.service` exists.
- Restarts only the companion connected-client collector around 03:15 local time each day, with up to 15 minutes of randomized delay.
- Uses `systemctl try-restart`, so an intentionally stopped or disabled collector is not started unexpectedly.
- Leaves Asterisk, MMDVM Bridge, Analog Bridge, Zello, and radio/bridge audio services untouched.
- Retains the Beta 5.6 Socket.IO cleanup repair and the 128 MB/192 MB systemd memory safeguards.
- Removes and disables the maintenance units on installations that do not have the companion collector.
- Updates the ASR main page and administration pages to display `v1.0.0 Beta 5.7`.

## Verified reason for the maintenance restart

On KE7WIL, the repaired collector ran approximately 21.5 hours with only 2-4 threads, 0.4 percent CPU, no new TGIF session errors, no systemd restarts, and no memory-pressure or OOM events. Its retained memory nevertheless rose from approximately 32 MB after restart to 89 MB. A manual collector-only restart returned it to 32 MB without restarting Asterisk or bridge-audio services.

## Reinstall

```bash
set -e
url="https://github.com/ke7wil-bridge/allscan-reimagined/releases/download/v1.0.0-beta.5.7/allscan-reimagined-1.0.0-beta.5.7.tar.gz"
pkg="/tmp/allscan-reimagined-1.0.0-beta.5.7.tar.gz"
expected="6f7805b29c4b60a3783bb803d2bda080a0f59bc649c5e68366d0c0587e71d461"
stage="/tmp/asr-beta-5.7-install"

curl -fL "$url" -o "$pkg"
printf '%s  %s\n' "$expected" "$pkg" | sha256sum -c -
rm -rf "$stage"
mkdir -p "$stage"
tar -xzf "$pkg" -C "$stage"
cd "$stage/allscan-reimagined-1.0.0-beta.5.7"
php -l payload/server/asr-api.php
php -l payload/scripts/asr-bridge-clients.php
sh -n payload/scripts/asr-asterisk-read.sh
bash -n payload/scripts/asr-reapply.sh
bash -n payload/scripts/asr-integrity-check.sh
python3 payload/scripts/asr-patch-connected-clients.py --self-test
bash ./install.sh
```
