# AllScan Reimagined v1.0.0 Beta 5.3

Beta 5.3 corrects Connection Status timer behavior while preserving the Beta 5 performance improvements.

## What Changed

- Connection Status timers now use the authoritative Asterisk SawStat values instead of a browser-generated timer.
- Timer updates are delivered every Asterisk polling cycle.
- Short unkey/rekey transitions are captured using David AllScan's 500 ms polling cadence during activity and for three seconds afterward.
- Adaptive slower polling remains active while the node is idle.
- Structural connection updates preserve the most recent valid timer values.
- HTML non-breaking-space placeholders are no longer displayed as timer text.
- Includes the Beta 5.2 faster post-refresh status population and automatic rollback-backup cleanup.

## Install

Copy this complete block into an interactive terminal on the AllStar node. It downloads the release, verifies the exact SHA-256 checksum, extracts it, lints the important files, and starts the installer:

```bash
set -e

url="https://github.com/ke7wil-bridge/allscan-reimagined/releases/download/v1.0.0-beta.5.3/allscan-reimagined-1.0.0-beta.5.3.tar.gz"
pkg="/tmp/allscan-reimagined-1.0.0-beta.5.3.tar.gz"
stage="/tmp/allscan-reimagined-install-beta5.3"
sum="8fafdde8f9fea29359cef54ad03aa2fb6669895f850fae4fef11e7519dd9e5a7"

sudo rm -rf "$stage"
sudo rm -f "$pkg"
curl -fL "$url" -o "$pkg"
echo "$sum  $pkg" | sha256sum -c -

mkdir -p "$stage"
tar -xzf "$pkg" -C "$stage"

cd "$stage/allscan-reimagined-1.0.0-beta.5.3"

php -l payload/server/asr-api.php
php -l payload/compat/allscan-v1.01/include/common.php
php -l payload/compat/allscan-v1.01/asr-settings/index.php
php -l payload/compat/allscan-v1.01/performance/index.php
php -l payload/compat/allscan-v1.01/lookup/index.php
php -l payload/compat/allscan-v1.01/echolink-lookup/index.php
php -l payload/compat/allscan-v1.01/astapi/server.php
php -l payload/compat/allscan-v1.01/astapi/AMI.php
php -l payload/scripts/asr-bridge-clients.php

sh -n payload/scripts/asr-asterisk-read.sh
bash -n payload/scripts/asr-reapply.sh
bash -n payload/scripts/asr-integrity-check.sh

bash ./install.sh
```

Do not wrap the final installer in a heredoc or other non-interactive shell. The installer and the official AllScan updater require an interactive terminal.
