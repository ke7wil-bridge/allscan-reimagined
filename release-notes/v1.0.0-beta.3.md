# AllScan Reimagined v1.0.0 Beta 3

This is a beta release for testing.

## Install

Run this as `root` on the node:

```bash
bash <<'ASR'
set -e

url="https://github.com/ke7wil-bridge/allscan-reimagined/releases/download/v1.0.0-beta.3/allscan-reimagined-1.0.0-beta.3.tar.gz"
pkg="/tmp/allscan-reimagined-1.0.0-beta.3.tar.gz"
stage="/tmp/allscan-reimagined-install-beta3"
sum="d1f27151fa37daf2e4ed675010c451b9f20f34fdb8fe99c7638bd31f299dd29f"

rm -rf "$stage"
curl -fL "$url" -o "$pkg"
echo "$sum  $pkg" | sha256sum -c -

mkdir -p "$stage"
tar -xzf "$pkg" -C "$stage"

cd "$stage/allscan-reimagined-1.0.0-beta.3"
./install.sh
ASR
```

## Header Title

During install, you will be asked for a **Header title**.

The Header title is the large title shown at the top of the AllScan Reimagined page.

Examples:

```text
KE7WIL | Node 641890
Netoholics Network
East Coast Reflector Monitor
W7ABC Home Node
Desert Link System
```

If you press Enter/Return without typing a Header title, it will default to:

```text
CALLSIGN | Node NODE_NUMBER
```

For example, if your node is `641890` and your callsign is `KE7WIL`, the default will be:

```text
KE7WIL | Node 641890
```

The browser tab title is created automatically from the Header title:

```text
Header title | ASR
```

For example:

```text
Netoholics Network | ASR
```

## Logo

You will also be asked for an optional logo path.

Press Enter/Return to use the default ASR logo. That is the recommended choice for beta testing.

Custom logo support is prepared for PNG, JPEG, and WebP images. The current beta accepts a logo file path on the node.

A friendlier Reimagined Settings page is planned for the Admin menu. It will let you upload a logo from your desktop, change the header title, and enable or disable bridge cards.

## What It Preserves

AllScan Reimagined installs or updates the official AllScan backend first, then applies the Reimagined interface and security layer.

Existing AllScan users, passwords, Favorites, node settings, and permissions are preserved.
