# AllScan Reimagined v1.0.0 Beta 5.2

Beta 5.2 is a small performance and installer-maintenance patch for Beta 5.

## What Changed

- Connection status now populates immediately after a browser refresh while retaining the single-feed load optimization.
- Old ASR rollback backups are pruned automatically without an approval prompt.
- The installer keeps the newest 10 rollback backups by default and removes only older timestamped ASR backup directories.

## Install

Copy this complete block into an interactive terminal on the AllStar node. It downloads the release, verifies the exact SHA-256 checksum, extracts it, and starts the installer:

```bash
set -e

url="https://github.com/ke7wil-bridge/allscan-reimagined/releases/download/v1.0.0-beta.5.2/allscan-reimagined-1.0.0-beta.5.2.tar.gz"
pkg="/tmp/allscan-reimagined-1.0.0-beta.5.2.tar.gz"
stage="/tmp/allscan-reimagined-install-beta5.2"
sum="ce3e76ef7a3f7e4faa26fe125273be6093ea96b4a579b4a6f31aa0f0af11e33b"

sudo rm -rf "$stage"
sudo rm -f "$pkg"
curl -fL "$url" -o "$pkg"
echo "$sum  $pkg" | sha256sum -c -

mkdir -p "$stage"
tar -xzf "$pkg" -C "$stage"

cd "$stage/allscan-reimagined-1.0.0-beta.5.2"
bash ./install.sh
```

Do not wrap the final installer in a heredoc or other non-interactive shell. The installer and the official AllScan updater require an interactive terminal.
