# AllScan Reimagined v1.0.0 Beta 5.5

Beta 5.5 corrects the Connection Status summary using the complete linked-node topology reported by Asterisk.

## What Changed

- Direct connections use Asterisk `RPT_NUMALINKS`, with the visible direct rows as a fallback.
- Total linked is the unique union of Asterisk `LinkedNodes` and the visible direct rows. This includes propagated nodes while also counting named IAX, Web Transceiver, and EchoLink clients that are not present in the numeric `LinkedNodes` list.
- Adjacent connections are calculated as total linked minus direct, without double-counting numeric direct nodes.
- The summary reads `X total linked (Y direct, Z adjacent)`.
- No Asterisk connection behavior, ASTAPI values, bridge monitoring, or public status-page code is changed.

## Install

Copy this complete block into an interactive terminal on the AllStar node. It downloads the release, verifies the exact SHA-256 checksum, extracts it, lints the important files, and starts the installer:

```bash
set -e

url="https://github.com/ke7wil-bridge/allscan-reimagined/releases/download/v1.0.0-beta.5.5/allscan-reimagined-1.0.0-beta.5.5.tar.gz"
pkg="/tmp/allscan-reimagined-1.0.0-beta.5.5.tar.gz"
stage="/tmp/allscan-reimagined-install-beta5.5"
sum="03e41f6fb6d33ddf630ba5655c53fca7c15ad88c73fed9a643ccce8d280594aa"

sudo rm -rf "$stage"
sudo rm -f "$pkg"
curl -fL "$url" -o "$pkg"
echo "$sum  $pkg" | sha256sum -c -

mkdir -p "$stage"
tar -xzf "$pkg" -C "$stage"

cd "$stage/allscan-reimagined-1.0.0-beta.5.5"

php -l payload/server/asr-api.php
php -l payload/compat/allscan-v1.01/include/common.php
php -l payload/compat/allscan-v1.01/asr-settings/index.php
php -l payload/compat/allscan-v1.01/performance/index.php
php -l payload/compat/allscan-v1.01/lookup/index.php
php -l payload/compat/allscan-v1.01/echolink-lookup/index.php
php -l payload/compat/allscan-v1.01/astapi/server.php
php -l payload/compat/allscan-v1.01/astapi/AMI.php
php -l payload/scripts/asr-bridge-clients.php

sh -n payload/scripts/asr-asterisk-read.sh
bash -n payload/scripts/asr-reapply.sh
bash -n payload/scripts/asr-integrity-check.sh

bash ./install.sh
```

Do not wrap the final installer in a heredoc or other non-interactive shell. The installer and the official AllScan updater require an interactive terminal.
