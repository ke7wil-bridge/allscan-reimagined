# AllScan Reimagined v1.0.0 Beta 5

This is the Beta 5 release of AllScan Reimagined, customized by KE7WIL and based on AllScan by David Gleason, NR9V.

## Install

Copy this complete block into an interactive terminal on the AllStar node. It downloads the release, verifies the exact SHA-256 checksum, extracts it, and starts the installer:

```bash
set -e

url="https://github.com/ke7wil-bridge/allscan-reimagined/releases/download/v1.0.0-beta.5/allscan-reimagined-1.0.0-beta.5.tar.gz"
pkg="/tmp/allscan-reimagined-1.0.0-beta.5.tar.gz"
stage="/tmp/allscan-reimagined-install-beta5"
sum="cb73a528f36d7a99417305bc1c63f65a6e7c9cedf8c27a4a61e28ac5a21a2cf8"

sudo rm -rf "$stage"
sudo rm -f "$pkg"
curl -fL "$url" -o "$pkg"
echo "$sum  $pkg" | sha256sum -c -

mkdir -p "$stage"
tar -xzf "$pkg" -C "$stage"

cd "$stage/allscan-reimagined-1.0.0-beta.5"
bash ./install.sh
```

Do not wrap the final installer in a heredoc or other non-interactive shell. The installer and the official AllScan updater require an interactive terminal.

## What Is New

- Preserves existing AllScan users, permissions, Favorites, node settings, and private databases during installation and updates.
- Adds Reimagined Settings, Performance Stats, lookup, EchoLink lookup, station-origin mapping, bridge diagnostics, and bug-report diagnostics.
- Adds responsive shared admin headers and menus for desktop, portrait, and landscape layouts.
- Improves connection counts, current-talker handling, bridge relay display, ASTAPI recovery, and stale bridge-client filtering.
- Adds QRZ-enriched station mapping with public-location fallback.
- Reduces load on smaller nodes with shared browser feeds, RAM caching, adaptive polling, hidden-tab handling, and low-power mode.
- Keeps custom branding in the top header while locking the footer to the ASR logo.
